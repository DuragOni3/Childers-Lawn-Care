# Childers Lawn & Landscape LLC — Website

Next.js 14 website for Childers Lawn & Landscape LLC.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
